# -
Foodability是一款基于Vue 3框架和Node.js包管理器的综合性食谱、食材、养生健康应用程序。该应用 旨在为用户提供一站式的食谱搜索、食材查看、烹饪建议以及生活健康信息。Foodability整合了多个API，包括关 键字搜索菜谱、获取菜谱详细信息、搜索食物、获取食物详细营养成分、健康新闻和养生知识，为用户提供全面的 食品与健康管理服务。

技术栈
前端框架: Vue 3
后端: Node.js
包管理器: Node.js/npm
